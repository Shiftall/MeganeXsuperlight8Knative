# CustomHeadsetGUI

## Start Development
1. clone repository
2. install [node.js](https://nodejs.org/) & [rust](https://www.rust-lang.org/tools/install)
3. run `npm i` in `CustomHeadsetGUI`
4. run `npm start` in `CustomHeadsetGUI` to start dev server

## Build
1. clone repository
2. install [node.js](https://nodejs.org/) & [rust](https://www.rust-lang.org/tools/install)
3. run `npm i` in `CustomHeadsetGUI`
4. run `npm build` in `CustomHeadsetGUI`
